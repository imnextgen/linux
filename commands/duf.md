# duf Command

[`duf`](https://github.com/muesli/duf) is a disk usage/free utility with pretty output.

## Installation

```bash
brew install duf
```

## Basic Usage

```bash
duf
```

## Common Options

- `duf -h` : Show help
- `duf -only local` : Show only local filesystems
- `duf -hide network,loops` : Hide network and loop filesystems
